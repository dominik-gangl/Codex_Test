# Codex_Test

Testumgebung für CODEX

## Simple Webapp

Dieses Repository enthält eine kleine Express-Webanwendung.

### Starten

1. Abhängigkeiten installieren
   ```bash
   npm install
   ```
2. Anwendung starten
   ```bash
   npm start
   ```

Die Anwendung läuft anschließend auf http://localhost:3000
